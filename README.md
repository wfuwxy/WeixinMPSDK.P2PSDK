功能介绍
=====================

用于微信公众账号获取用户详细信息及头像、获取媒体文、绑定OpenId-Fakeid、自动推动消息等。

P2P严正声明：我们致力于提供更加优秀的用户实时互动体验，反对使用P2P进行群发骚扰用户。
